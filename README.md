# DevArtech Marketplace

A Claude Code plugin marketplace for development tools and automation.

## Installation
To install the marketplace as a whole, while in Claude Code, run:
```bash
/plugin marketplace add https://github.com/DevArtech/devartech-marketplace
```


To install plugins from this marketplace:

```bash
claude plugins add devartech-marketplace:<plugin-name>
```

## Usage

List available plugins:

```bash
claude plugins list devartech-marketplace
```

Install a specific plugin:

```bash
claude plugins add devartech-marketplace:n8n-compliance
```
